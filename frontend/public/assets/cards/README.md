# Card Graphics

Add your custom card PNG/JPG files here.

## Structure

Create folders for each suit:
- `hearts/` - A.png, 2.png, 3.png, ..., K.png
- `diamonds/` - A.png, 2.png, 3.png, ..., K.png
- `clubs/` - A.png, 2.png, 3.png, ..., K.png
- `spades/` - A.png, 2.png, 3.png, ..., K.png
- `back.png` - Card back design

## Card Dimensions

Recommended: 200px × 300px (2:3 aspect ratio)
Minimum: 100px × 150px

## Format

PNG with transparency or JPG

## Total Cards

52 card images + 1 back design = 53 files

## Example

```
cards/
├── hearts/
│   ├── A.png
│   ├── 2.png
│   ├── ...
│   └── K.png
├── diamonds/
│   ├── A.png
│   └── ...
├── clubs/
│   ├── A.png
│   └── ...
├── spades/
│   ├── A.png
│   └── ...
└── back.png
```
